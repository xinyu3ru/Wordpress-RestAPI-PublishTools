### 安装 vps 服务器

我的工具网站是建立在一个 vps 服务器上，购买服务器之后还需要安装管理面板，我一般喜欢宝塔面板，虽然现在商业化越来越严重，但是目前还没有替代的管理面板。
它的 Ubuntu 安装代码是：

```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec
```
